# ⌨️ TypeMaster Pro

A modern Typing Speed Test desktop application built using Python and CustomTkinter.

## Features

- 🚀 Real-time Typing Speed (WPM)
- 🎯 Live Accuracy
- ❌ Mistakes Counter
- 📈 Progress Bar
- 🔀 Random Sentences
- 💻 Modern User Interface
- 🧩 Object-Oriented Programming

## Technologies Used

- Python
- CustomTkinter
- OOP

## Run

```bash
pip install customtkinter
python main.py
```
